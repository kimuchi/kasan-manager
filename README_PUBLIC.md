# 加算マネージャ（公開資料）

> 介護保険・障害福祉サービスの **加算（報酬加算）取得を AI が支援する** Web ツールです。
> 個別事業所のデータをアップロード／CPOS 連携で取り込み、取り漏れの可能性が高い加算と「どうすれば取れるか」を JSON / Markdown で返します。

---

## 1. 何ができるか

- 加算マスタと事業所データを照合して、**取得可能性の高い加算**を提案
- AI（Gemini）で **「どうすれば取れるか」のアクション案** を生成
- 介護給付費明細書 PDF から算定中加算を自動抽出
- 介護保険・医療保険・障害福祉の主要サービスに対応

> ⚠️ 加算算定可否を法的に保証するものではありません。最終判断は所管自治体・社労士・行政書士へ。

## 2. 利用方法（3 つ）

### A. CPOS 連携（推奨）

CPOS（介護記録・請求基盤）にデータがある場合、**自分の CPOS PAT** をブラウザに入力するだけで、CPOS から事業所・利用者・職員・請求の集計データを取得して分析できます。

- PAT は加算マネージャの DB に保存されません
- 暗号化 HTTP-only Cookie だけがあなたのブラウザに残ります
- 操作は [CPOS PAT 接続ガイド](/docs/CPOS_TOKEN.md) を参照

### B. JSON / PDF 直接アップロード

CPOS を使わず、自分で用意した `tenant_status.json` / `staff.json` / `user_summary.json` / レセプト PDF をアップロードして分析できます。

- アップロードファイルはサーバには保存しません（メモリ処理のみ）
- 個人情報を含めないでください（[データ取扱方針](/docs/DATA_SAFETY.md)）
- フォーマットの詳細は [JSON 形式リファレンス](/docs/json/) を参照

### C. Web フォーム入力

事業所情報と気になっている加算をフォームで自由記述するだけでも、Gemini ベースの提案が受け取れます。デモ・お試し用途に。

## 4. プライバシー・セキュリティ

| | |
|---|---|
| PAT の保存 | サーバ DB / ファイル / 環境変数には保存しない（暗号化 Cookie のみ） |
| アップロード PDF | メモリ処理のみ・サーバ保存なし |
| 個人情報 | 既定で集計値のみ。フォーム入力もマスキング推奨 |
| 通信 | 全 HTTPS（本番） |
| アクセス制御 | レート制限（IP 単位）・reCAPTCHA v3（任意） |

詳細は [DATA_SAFETY.md](/docs/DATA_SAFETY.md)。
