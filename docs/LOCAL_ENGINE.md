# ローカル前処理エンジン（ブラウザ完結）

フォルダを選ぶだけで、PC の中にある書類（介護給付費明細書 PDF・利用者一覧 Excel・勤務表など）を
**ブラウザの中だけ**で解析し、加算チェックに必要な集計値だけをクラウドに送る機能です。

ページ: [`/`](/)（公開トップ。旧 URL `/local-import` も利用可）

このローカル版は **ログイン不要・無料** で使えます。生成AIによる取り方提案・CPOS 連携・
判定履歴/レビュー・ポートフォリオ最適化などの **高精度版**（有料・要ログイン）は [`/pro`](/pro)、
お申し込み・詳細は [ケア・プランニング株式会社のサービス案内](https://www.care-planning.co.jp/systems/kasan/)
をご覧ください。近日、**デスクトップアプリ**からの利用にも対応予定です。

---

## いちばん大事なこと：生データはブラウザから出ません

- PDF・Excel・CSV・画像の**読み取り（OCR）・分類・個人情報の除去・集計**は、すべてあなたのブラウザのタブの中で行います。
- クラウドへ送るのは「**件数・比率・要件フラグ**」だけです。氏名・被保険者番号・住所・電話・生年月日などは送りません。
- 送信の前に、**実際に送られる JSON を画面に全文表示**します（ドライラン）。中身を確認してから送信してください。

---

## 使い方

1. 上部のプルダウンで対象サービス（自動判定も可）と対象月を確認します。
2. 「📁 フォルダを選ぶ」を押し、書類の入ったフォルダを選びます。
   - Chrome / Edge ではフォルダごと選べます。それ以外のブラウザでは複数ファイル選択になります。
3. 各ファイルが自動で分類・集計されます（「📄 解析状況」に表示）。
4. 「🔍 送信されるデータ（ドライラン）」で中身を確認します。
5. 「この内容で加算チェックを実行」を押すと、集計値だけが送られ、結果が表示されます。

スキャン PDF や写真は、「OCR で読む」にチェックすると読み取れます（初回のみ言語データをダウンロードするため時間がかかります）。

## 対応している書類と、抽出する集計値

| 書類 | 主な形式 | 抽出する集計値（個人情報は捨てます） |
|---|---|---|
| 介護給付費明細書 / レセプト | PDF | 算定中の加算の件数、要介護度分布、サービスコード |
| 利用者一覧 / 受給者台帳 | Excel / CSV | 要介護度別の人数、要介護3以上の比率（**氏名・被保険者番号の列はまるごと破棄**） |
| 勤務形態一覧 / 職員名簿 | Excel / CSV | 職種別の有資格者数・常勤換算（**氏名の列は破棄**） |
| 体制等状況一覧 / 届出 | Excel / PDF | 参考情報 |

表形式（Excel/CSV）は「必要な列だけを読む」方式です。氏名・被保険者番号・住所などの列は**そもそも読み込みません**。
個別の行は集計したあとに破棄します。

## 個人情報（PII）の扱い

二重の防御で個人情報を残しません。

1. **列ドロップ（表形式）**: 氏名・カナ・被保険者番号・住所・電話・生年月日などの列を、読み込み対象から外します。
2. **伏字化（テキスト）**: PDF/OCR から得た文章中の被保険者番号・電話番号・メール・生年月日などを伏字化します。
3. **送信前チェック**: 完成した集計バンドルに数値系の個人情報が残っていないか走査し、見つかれば送信を中止します。

詳しくは [データ取扱方針](/docs/DATA_SAFETY.md) もご覧ください。

## クラウド側の処理

送られた集計バンドルは、CPOS 連携と同じ判定経路で処理されます。

- 受け口: `POST /api/analyze/from-local`
- 形式: `analysis_source`（schemaVersion 1.0）互換。レセプト由来の加算は `claimEvidence`（kasan_key ベース）として同梱され、そのまま判定の根拠に使われます。
- 系統: 解析結果は `source_type = local_engine` として記録されます。
- 認証: このローカル版は無認証（無料枠）でご利用いただけます。判定履歴の保存・レビュー・AI 補完などの高精度版は有料（要ログイン、[`/pro`](/pro)）です。

## 対応ブラウザ・制限事項

- フォルダ選択は PC 版の Chromium 系ブラウザ（Chrome / Edge / Brave / Opera）で利用できます。Firefox / Safari は未対応のため複数ファイル選択にフォールバックします。
- スマートフォン・タブレット（iOS Safari・Android Chrome 等）はフォルダ選択に未対応です。PC をご利用ください。HTTPS（セキュアコンテキスト）必須です。
- CSV は Shift-JIS（CP932）/ UTF-8 の自動判定を試みます。Excel で開く前提なら UTF-8（BOM 付き）が安全です。スキャン画像 PDF はテキスト層が無いため OCR が必要で低速・精度低下します。
- レセプトからのサービスコード抽出は社内資料・大臣基準告示に基づくパターンマッチで、公式サービスコード表との完全一致は未検証です。帳票形式により抽出精度が変動します。
- OCR・大きな PDF の処理は端末の性能・メモリに依存します。
- 現在の対応サービスは 通所介護 / 訪問介護 / 訪問看護（介護保険）/ 居宅介護支援 です。
