🗂 解析履歴（直近 50 件）

📋 解析詳細

各加算ごとに「承認」「差戻し」「保留」を記録できます。すべて承認になれば解析全体も自動的に承認状態に集約されます。

あと一歩で取れる加算を、月額収益（概算）と必要工数で並べた優先度ランキングです。 地域単価・実稼働日数の前提は PORTFOLIO.md を参照してください。

加算マスタの整合性レビュー結果（alpha.5.9〜.5.13 packets）。 現在の解析対象サービスを中心に、加算マスタの三層コードモデル （public/official / receipt detection / internal legacy）と推奨初期判断を表示します。

          🔐
          CPOS と連携して分析する（任意・クリックで開く）
          手順
        

        
        
        
          
        
            あなた個人の CPOS API トークン（cpos_pat_...）を入力すると、
            あなたの権限内の事業所データを CPOS から取得して分析できます。
          
        

          
          
            
        1. 
              CPOS にログイン（Google アカウント）して
              PAT（個人トークン）画面
              を開く
              ※ CPOS URL を下に入力するとリンクが自動で有効になります
            
          2. 
            
        2. 加算マネージャ用の個人 API トークン（PAT）を発行して、表示された cpos_pat_... をコピー
          3. 
            
        3. 下の「CPOS API トークン」欄に貼り付けて「保存して接続確認」を押す
          4. 
          
        

          
          
            
        • 🔒 トークンはこのアプリの DB に保存されません
          • 
            
        • 🔒 暗号化された HTTP-only Cookie としてあなたのブラウザにのみ保存されます
          • 
            
        • 🔒 ブラウザの localStorage / sessionStorage は使いません
          • 
            
        • 🔒 CPOS へのアクセスはサーバ間でのみ実施。ブラウザから直接 CPOS を叩きません
          • 
            
        • 👉 詳しくは CPOS PAT 接続ガイド /
              データ取扱方針

        1. 分析するサービスを選ぶ
