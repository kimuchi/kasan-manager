🔐 CPOS と連携して分析する（任意）

あなた個人の CPOS API トークン（cpos_pat_...）を入力すると、 あなたの権限内の事業所データを CPOS から取得して分析できます。

  • 🔒 トークンはこのアプリの DB に保存されません
  • 🔒 暗号化された HTTP-only Cookie としてあなたのブラウザにのみ保存されます
  • 🔒 ブラウザの localStorage / sessionStorage は使いません
  • 🔒 CPOS へのアクセスはサーバ間でのみ実施。ブラウザから直接 CPOS を叩きません
  • 👉 詳しくは CPOS PAT 接続ガイド / データ取扱方針

  • トークン:
  • 権限: -
  • 事業所範囲: -
  • 有効期限: -

1. 分析するサービスを選ぶ

介護保険の通所介護・訪問介護のほか、障害福祉の就労継続支援などにも対応しています。

サービスを選ぶと、対応する加算マスタを参照して分析します。

2. 事業所の情報を入力する

分かる範囲で OK です。空欄でも分析できますが、入力するほど精度が上がります。

3. ファイルをアップロード（任意）

レセプトPDFや、職員/利用者集計の JSON を添付すると、決定的判定が動きます。

    ⚠️ 個人情報（被保険者番号・氏名・住所）はアップロード前にマスキングしてください。本ツールはその場で分析のみを行い、ファイルは保存しません。

    4. 分析を実行する

    「判定 + AI」: 加算マスタ判定 + Gemini で取り方アクションを生成（10〜30秒）
    「判定エンジンのみ」: マスタ + DSL のみで素早く判定（数秒）
